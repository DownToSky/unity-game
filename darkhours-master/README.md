# darkhours

TopDown Prototyp Shooter (with endless rooms)

"It's unfortunately only a prototype. Gameplay elements (enemys, objectives and collectibles) will be added later."

## CONTROLS
[W A S D] - movement

[Left Mouse Button] - shoot

[Mouse] - looking around
